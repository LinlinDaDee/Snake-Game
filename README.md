# Snake-Game
## Overview
This project is a classic Snake Game implemented using JavaScript, HTML, and CSS. The game involves controlling a snake to eat food that appears on the screen, causing the snake to grow in length. The goal is to avoid running into the walls or the snakeâ€™s own body.
## Running the Game
1.	Open index.html in your web browser.
2.	Use the arrow keys to control the snake:
- Up Arrow: Move up
- Down Arrow: Move down
- Left Arrow: Move left
- Right Arrow: Move right
